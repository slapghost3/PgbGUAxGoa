# 项目简介

## 核心功能
- 用户管理
  - 用户注册添加
  - 用户账号删除
  - 用户信息修改
  - 用户列表查询
  - 用户详情查询
- 论文题目管理
  - 题目发布添加
  - 题目信息删除
  - 题目内容修改
  - 题目检索查询
  - 题目详情查询
- 知识图谱管理
  - 节点关系创建
  - 图谱数据删除
  - 图谱结构更新
  - 知识关联查询
- 文本关键词抽取
  - 论文题目推荐
- 选题过程管理
  - 选题流程创建
  - 过程记录删除
  - 选题状态更新
  - 进度状态查询
  - 选题记录查询
- 可视化分析
  - 研究方向可视化展示
  - 选题热度分析
  - 全局统计信息查询

## 适用场景
- 高等教育机构
- 研究机构
- 论文选题指导

## 技术特点
- 基于知识图谱技术，提供智能化的论文选题推荐
- 用户友好的界面设计，易于操作
- 多功能模块，满足不同用户需求
